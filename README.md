# 🏋️‍♂️ AI-Fit: 基于大模型的智能健身辅助系统

> **💡 项目简介**
> 
> 本项目为《XXX软件综合实践》小组项目。
> 
> **AI-Fit** 是一款专为健身新手和时间碎片化人群设计的智能健身辅助平台。针对传统健身应用“计划死板、缺乏专业指导”的痛点，本项目引入**大语言模型 (LLM)**，为用户提供动态的个性化训练计划与24小时AI私教答疑。同时，创新性地采用“AI意图识别+本地高质量动作库”的混合架构，在保证智能化的前提下，确保了动作指导的绝对专业性与安全性。

---

## ✨ 核心功能模块

本项目包含以下 5 大核心功能模块：

### 1. 🤖 AI 个性化计划生成器 (核心亮点)
- 用户输入身体数据、健身目标（减脂/增肌）、伤病史及本周可用天数。
- 后台通过 AI 大语言模型动态生成包含**训练动作、组数、饮食热量建议**的定制化计划，并在前端渲染为可视化任务卡片。

### 2. 📖 动作姿势指导库 (AI解析 + 本地规范库)
- 用户输入自然语言（如“大腿内侧怎么减”），AI 提取意图后，从本地数据库精准匹配**标准动作规范动图 (GIF)**。
- 由大模型实时生成详尽的“文字版”动作要领、呼吸节奏与发力提示，兼顾智能交互与动作安全性。

### 3. 💬 24小时 AI 健身教练
- 具备上下文记忆能力的垂直领域智能助手。
- 结合用户的个人健康档案（如体重、目标期），提供个性化、符合健身科学的饮食与训练答疑。

### 4. 🧑‍⚕️ 用户数据与健康档案
- 记录用户的身高、体重、目标，自动计算 BMI 及基础代谢率等健康指标。
- 支持伤病史录入，作为 AI 生成计划的强制安全约束条件。

### 5. 📅 训练与饮食打卡日历
- 每日任务打卡机制，以日历视图直观展示坚持天数与训练频次。
- 自动统计体重变化趋势，提升用户黏性与成就感。

#  菲特勒斯——你的个人健康管理专家需求规格说明书

## 1. 项目概述

菲特勒斯——你的个人健康管理专家 是一个基于AI驱动的个人健康管理工具，旨在通过快速记录、系统分析和智能建议，为用户提供个性化的健康管理和健身指导。该项目采用Next.js 15框架开发，集成了AI API进行智能解析和建议生成。

## 2. 用户需求分析

### 2.1 核心用户群体
- **个人健身爱好者**：需要记录和分析日常饮食、运动数据
- **健康管理用户**：关注体重、营养摄入和健康指标
- **AI技术用户**：希望获得智能化的健康建议和指导

### 2.2 用户痛点
- 传统健身追踪需要繁琐的手动数据录入
- 难以保持数据一致性和获得有意义的洞察
- 缺乏个性化的健康建议和指导
- 多维度健康数据（饮食、运动、心情、睡眠）难以整合分析

## 3. 功能需求

### 3.1 健康记录功能

#### 3.1.1 饮食记录
- **自然语言输入**：支持文本描述食物摄入
- **图片识别**：通过上传食物照片进行自动识别
- **营养成分解析**：自动解析食物的营养成分（卡路里、蛋白质、碳水化合物、脂肪等）
- **餐次分类**：自动识别早餐、午餐、晚餐、零食等餐次类型
- **摄入量记录**：记录食物消耗的克数

#### 3.1.2 运动记录
- **运动类型记录**：支持有氧运动、力量训练、柔韧性训练等
- **运动时长记录**：记录运动持续时间
- **卡路里消耗计算**：基于用户体重和运动类型计算消耗的卡路里
- **运动强度分析**：分析运动的MET值和肌肉群锻炼

#### 3.1.3 每日状态记录
- **心情记录**：记录每日心情状态（1-6级）
- **压力水平**：记录压力程度（1-6级）
- **健康状况**：记录整体健康感受（1-6级）
- **睡眠质量**：记录睡眠质量和时长
- **补充说明**：支持文字描述详细情况

### 3.2 数据分析功能

#### 3.2.1 营养分析
- **总卡路里摄入**：计算每日总卡路里摄入量
- **宏量营养素分析**：分析碳水化合物、蛋白质、脂肪的摄入比例
- **微量营养素分析**：记录其他营养素摄入情况
- **TEF分析**：食物热效应分析，计算消化食物消耗的能量

#### 3.2.2 代谢计算
- **BMR计算**：基础代谢率计算
- **TDEE计算**：每日总能量消耗计算
- **体重变化趋势**：分析体重变化趋势
- **活动水平调整**：根据活动水平调整代谢计算

### 3.3 AI智能建议

#### 3.3.1 智能建议生成
- **基于历史数据分析**：根据7天历史数据生成建议
- **多维度分析**：结合饮食、运动、心情、睡眠等多维度数据
- **个性化建议**：根据用户目标和身体状况提供个性化建议
- **建议分类**：按优先级（高、中、低）分类建议

#### 3.3.2 AI角色系统
- **营养师角色**：提供营养建议
- **健身教练角色**：提供运动建议
- **健康顾问角色**：提供综合健康建议
- **动态记忆**：AI能够记住用户的健康信息和偏好

### 3.4 用户配置功能

#### 3.4.1 个人信息设置
- **基本身体信息**：体重、身高、年龄、性别
- **活动水平**：选择日常活动水平（久坐、轻度、中度、活跃、非常活跃）
- **健康目标**：维持、减重、增重等目标
- **BMR计算公式**：支持Mifflin-St Jeor和Harris-Benedict公式

#### 3.4.2 专业模式
- **健康档案**：记录疾病史、过敏史、药物使用情况
- **生活方式**：记录饮食偏好、睡眠习惯、压力水平
- **健康认知**：记录用户对健康的认知和期望

### 3.5 数据可视化

#### 3.5.1 图表展示
- **营养摄入趋势**：展示每日营养摄入变化
- **体重变化趋势**：展示体重变化曲线
- **卡路里平衡**：展示摄入与消耗的平衡情况
- **TEF分析图表**：展示食物热效应分析结果

#### 3.5.2 数据导出
- **数据备份**：支持数据导出和备份
- **导出提醒**：定期提醒用户备份数据
